# Kurapati Saiteja | Portfolio v2

A modern, highly interactive personal portfolio designed to showcase my full-stack engineering and UI/UX design capabilities. Built with a focus on premium typography, smooth micro-animations, and fluid responsiveness.

### 🚀 Tech Stack
* **Framework:** React.js + Vite
* **Animations:** GSAP (ScrollTrigger, Timelines)
* **Smooth Scrolling:** Lenis
* **Styling:** Vanilla CSS (CSS Variables, Flexbox/Grid, Clamps for dynamic scaling)

### ⚡ Key Features
* Custom collision-flash preloader (Desktop optimized).
* Fully responsive fluid typography using CSS `clamp()`.
* High-performance scroll-linked animations.
* Custom cursor integration.
